## rollup 打包配置

### rollup.config.js

- 配置入口和出口文件
- 采用 umd 打包方式挂载 global.Vue
- 采用 .babelrc 的 @babel/preset-env 预设插件

### package.json 配置

- 运行 npm run dev 执行 rollup 打包 -c:指定配置文件 -w:监听文件变化

## 初始化数据

### 创建 Vue 构造函数

- 传入用户选项 options
- 调用初始化方法 \_init

### initMixin 扩展 init 方法

- 将用户选项挂载在实例上
- 初始化状态 initState
- initData 初始化 data
