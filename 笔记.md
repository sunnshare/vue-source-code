## rollup 打包配置

### rollup.config.js

- 配置入口和出口文件
- 采用 umd 打包方式挂载 global.Vue
- 采用 .babelrc 的 @babel/preset-env 预设插件

### package.json 配置

- 运行 npm run dev 执行 rollup 打包 -c:指定配置文件 -w:监听文件变化
